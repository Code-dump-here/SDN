# Question Bank Management Application (Assignment 2)

This project is a self-contained Node.js Express application for managing quizzes and questions, with both API and UI in a single server. It uses EJS and Handlebars for views, Bootstrap for styling, and MongoDB (via Mongoose) for data storage.

## Features
- CRUD for quizzes and questions
- Add questions to quizzes
- View quiz details with related questions
- Responsive UI with Bootstrap
- All API and UI routes in one server

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start MongoDB (local or Atlas)
3. Run the app:
   ```bash
   node app.js
   ```
4. Visit http://localhost:3000 in your browser

## Folder Structure
- `/routes` - Express routes for API and UI
- `/views` - EJS/Handlebars templates
- `/models` - Mongoose models
- `/public` - Static assets (CSS, JS)

## Requirements
- Node.js
- MongoDB
